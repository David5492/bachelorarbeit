============
Installation
============


Stable release
--------------

To install  Climate Change Mitigation, run this command in your terminal:

.. code-block:: console

    $ pip install _climate_change_mitigation

This is the preferred method to install  Climate Change Mitigation, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for  Climate Change Mitigation can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/david5492/_climate_change_mitigation

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/david5492/_climate_change_mitigation/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/david5492/_climate_change_mitigation
.. _tarball: https://github.com/david5492/_climate_change_mitigation/tarball/master
